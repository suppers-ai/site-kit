<sa-feature-grid> with <sa-feature> children

Auto-fitting CSS grid (minmax(280px, 1fr)). Children typically <sa-feature> instances but anything works — the grid doesn't introspect children.

Each component is a single ES module under dist/components/. A page imports only what it uses; nothing else loads.

 🎨

Override --sa-accent on :root and every component picks it up automatically through CSS-custom-property inheritance.

Slots collapse cleanly when nothing is provided. This card has no icon slot — the layout adjusts without empty space.
