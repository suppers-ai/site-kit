.sa-prose demo

How prose flows in site-kit

Long-form text picks up its rhythm from .sa-prose: vertical spacing scales with heading level, lists indent further than top-level paragraphs, and code blocks adopt the dark surface treatment without manual styling.

What gets styled

Direct children of .sa-prose get the rhythm rules. Nested elements (a list inside a blockquote, say) keep their own spacing, so reusable widgets dropped into prose don't fight the article rhythm.

Override surface

Sites override --sa-accent on :root after loading the kit. Hover and focus states derive automatically — no per-site recolouring.

:root { --sa-accent: #fe6627; }   /* one declaration */
  • Tokens are the contract.
  • Element styles are the defaults.
  • Utilities (.sa-container, .sa-prose) are the only kit-shipped class names.
