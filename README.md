# site-kit

Shared design system and component library for Suppers AI sites.

Currently ships:

- `dist/design-system.css` — design tokens, reset, typography.
- `fonts/` — vendored Itim and JetBrains Mono.
- `showcase/index.html` — live demo of the foundation.

See `docs/superpowers/specs/2026-04-29-site-kit-foundation-design.md`
in the parent workspace for the design intent.
